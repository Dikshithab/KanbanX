# рҹҡҖ Installation

## Clone Repository

```bash
git clone https://github.com/Dikshithab/KanbanX.git
```

## Backend

```bash
cd backend
./mvnw spring-boot:run
```

Backend runs on:

```
http://localhost:8080
```

---

## Frontend

```bash
cd frontend
npm install
npm run dev
```

Frontend runs on:

```
http://localhost:5173
```
